# スマホでマウス操作（PhoneMouse）

スマホが、パソコンの**マウス＆キーボードがわり**になるアプリです。
パソコンで起動して、出てきた **QRコードをスマホのカメラで読むだけ**。
あとは指でなぞってカーソルを動かしたり、文字を打ったりできます。

- スマホには**アプリのインストールは不要**（カメラとブラウザだけ。iPhone / Android どちらもOK）
- パソコンは **Mac / Windows** に対応

---

## 📱 はじめての方へ — かんたん使い方ガイド

> パソコンが苦手でも大丈夫。順番にやれば使えます。

### ✅ 用意するもの
- パソコン（Mac か Windows）
- スマートフォン
- **パソコンとスマホを「同じ Wi-Fi」につなぐ** ← ここが一番大事です

### 🚀 使う手順

**① パソコンでアプリを起動する**
受け取った **PhoneMouse**（Macは `PhoneMouse.app`、Windowsは `PhoneMouse.exe`）を **ダブルクリック**します。

**②（最初の1回だけ）「開けません」と出たときの対処**
このアプリはまだ“身元証明（署名）”を付けていないため、初回だけ警告が出ます。**こわれませんので安心してください。**
- **Mac**: アイコンを **右クリック →「開く」** → もう一度出る画面で **「開く」**。
- **Windows**: 「WindowsによってPCが保護されました」と出たら → **「詳細情報」→「実行」**。

**③ パソコンの画面にQRコードが出る**
起動すると、パソコンの**ブラウザが自動で開いてQRコードが表示**されます。
（もし出てこないときは、画面のはしっこ（Mac＝画面上のメニューバー／Windows＝右下）に出る**小さな丸いアイコン**をクリックして「QRを表示」を選んでください。）

**④ スマホのカメラでQRを読む**
スマホの**カメラアプリ**をQRコードに向けると、画面にリンクが出ます。それを**タップ**すると操作画面が開きます。

**⑤ 指で操作する！**
操作画面の広い部分（トラックパッド）を指でなぞると、パソコンのカーソルが動きます。🎉

---

### 🖐 指の使いかた（早見表）

| やりたいこと | 指の動き |
|---|---|
| カーソルを動かす | **1本指**でなぞる |
| 左クリック | **1本指**で軽くトン（タップ） |
| 右クリック | **2本指**で軽くトン |
| スクロール（上下に動かす） | **2本指**でなぞる |
| 拡大・縮小 | **2本指**を広げる／つまむ |
| ドラッグ（つかんで運ぶ） | **長押し** → そのまま指を動かす → 離す |
| 文字を入力する | 画面の **「⌨ キーボードを開く」** を押す |

**文字入力について**：「キーボードを開く」を押すとスマホのキーボードが出ます。打った文字がそのままパソコンに入ります（**日本語の変換にも対応**しています）。

---

### 🔑 Mac をお使いの方へ（最初の1回だけ・大事）

Macは安全のため、最初に「アプリがマウスを動かしてよい」という許可が必要です。
**カーソルが動かないとき**は、次の設定をしてください。

1. アプリを起動したまま、**`システム設定`** を開く
2. **`プライバシーとセキュリティ`** → **`アクセシビリティ`** を開く
3. 一覧の中の **PhoneMouse** を **オン（青）** にする
4. アプリを一度終了して、**もう一度起動**する

> Windowsの方は、この設定はいりません。最初に「ファイアウォールの許可」を聞かれたら **「アクセスを許可する」** を押してください。

---

### 😟 うまくいかないとき

- **スマホからつながらない**
  → パソコンとスマホが **同じ Wi-Fi** につながっているか確認してください。**いちばん多い原因**です。
- **カーソルが動かない（Mac）**
  → 上の「アクセシビリティ」の許可をして、アプリを再起動してください。
- **QRコードがうまく読めない**
  → QRコードの下に出ている文字（`http://…` で始まる住所）を、スマホのブラウザのアドレス欄に**手で打ち込んでも**つながります。
- **会社・学校・お店のWi-Fiでつながらない**
  → そういうWi-Fiは、端末どうしの通信を禁止していることがあります。その場合は下の「[どんな回線でも使う（Tailscale）](#どんな回線でも使うtailscale)」を見てください。

### 🛑 終わるとき
パソコンの画面のはしっこ（Mac＝上のメニューバー／Windows＝右下）にある**丸いアイコン**をクリック → **「終了」** を選びます。

## 🛠 ここから下は「作る人・くわしい人」向け

### ソースから動かす（開発）

> パソコンとスマホを**同じ Wi-Fi** につないでおくこと。

```bash
# 1) 依存をインストール（初回のみ）
python3 -m venv .venv
.venv/bin/pip install -r app/requirements.txt   # Windows は .venv\Scripts\pip

# 2) 起動
.venv/bin/python -m app.main                     # Windows は .venv\Scripts\python -m app.main
```

起動すると **PCのブラウザが自動で開き、QRコードと接続URL** が表示されます。常駐アイコン（メニューバー/タスクトレイ）から「QRを表示」「終了」ができます。

接続には**ペアリングのトークン**が必要です（QR/URLに自動で含まれます）。トークンを含まないURLや、他人が勝手に推測したURLでは操作できません。`/info`・`/qr.png` はPC本体（ローカル）からのみ取得できます。

### 配布用にパッケージ化（ダブルクリック起動）

PyInstaller で実行ファイルを作れます。**OSごとに、そのOS上で**ビルドしてください（クロスコンパイル不可）。

```bash
.venv/bin/pip install pyinstaller
# macOS
pyinstaller packaging/build_mac.spec      # → dist/PhoneMouse.app
# Windows
pyinstaller packaging/build_win.spec      # → dist/PhoneMouse.exe
```

ビルド後は **`dist/PhoneMouse.app`（Mac）/ `dist/PhoneMouse.exe`（Windows）をダブルクリック**するだけ。起動すると自動でブラウザにQRが出ます（メニューバー/タスクトレイ常駐で終了）。

- v1 は**未署名**です。配布相手のPCでは初回に警告が出ます:
  - macOS: Finder で「**右クリック → 開く**」で回避。
  - Windows: SmartScreen で「**詳細情報 → 実行**」。
- 警告を消すにはコード署名（Apple Developer ID / Windows コード署名証明書）が必要で、今後の課題です。

### どんな回線でも使う（Tailscale）

`KUDOS_SECURE_X` のような「端末間通信を禁止（AP/クライアント分離）」したWi-Fiでは、同じLANにいてもスマホ↔PCが通信できません。これを回避するには **Tailscale**（無料のメッシュVPN）を使います。

1. **PC（Mac/Windows）に Tailscale を入れてサインイン**（App Store / `brew install --cask tailscale` / tailscale.com）。
2. **スマホにも Tailscale アプリ**を入れ、**同じアカウント**でサインインして **ON**。
3. アプリ（PhoneMouse）を起動 → 表示されるQR/URLが **`http://100.x.x.x:8000`（Tailscale の IP）** になります。
4. スマホ（Tailscale ON）でそのQRを読む → **回線の種類やクライアント分離に関係なく接続**できます。

> 仕組み: サーバーは全インターフェースで待ち受けているので、Tailscaleを入れると 100.x の仮想IPでも届く。アプリはTailscaleのIPを自動検出してQRを出すので、両端末でTailscaleをONにするだけ。Tailscaleが無い/OFFのときは同じWi-Fi用のLAN IPを案内します。

### トラブルシューティング（技術）

- **カーソルが動かない（Mac）** → アクセシビリティ権限が未許可。設定を確認しアプリ再起動。
- **スマホからつながらない** → 同じWi-Fiか確認。ファイアウォールで Python/アプリの受信を許可。
- **QRのURLが `127.0.0.1`** → LAN IPを取得できていない。Wi-Fi接続を確認。
- **操作できない／すぐ切れる** → QR（トークン付きURL）から開き直す。トークン無しの接続は拒否されます。
